

# 🎬 Castify

**Castify** is an interactive platform where users can vote on actors for book-to-movie adaptations. It brings fan opinions into the casting conversation and helps visualize the most popular choices.

## 🚀 Features

* 🗳️ Vote for actors for each character
* 📊 View live voting trends
* 🎭 Multiple genres (Romance, Fantasy, Sci-Fi, Drama)
* ⚡ Clean, modern UI
* 📱 Responsive landing page

---

## 💡 Idea

Castify allows fans to:

* Choose their dream cast for book adaptations
* Influence casting trends
* Engage with upcoming movie concepts
